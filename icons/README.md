# Icons

Place PNG icons here at the following sizes:

- `icon16.png`  — 16×16px  (toolbar / favicon)
- `icon48.png`  — 48×48px  (extensions management page)
- `icon128.png` — 128×128px (Chrome Web Store)

The hex symbol ⬡ from the Sidebar logo works well as the base shape.
Recommended: use a filled ⬡ on a white or transparent background in `#1a73e8` (Google Blue).
